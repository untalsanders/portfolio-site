Portfolio Blog Starter

Esta es una plantilla de sitio de portafolio con blog incluido. Incluye:

  • Compatibilidad con MDX y Markdown
  • Optimizado para SEO (mapa del sitio, robots, esquema JSON-LD)
  • Fuente RSS
  • Imágenes OG dinámicas
  • Resaltado de sintaxis
  • Tailwind v4
  • Análisis de velocidad de Vercel / Análisis web
  • Fuente Geist

Demostración

https://portfolio-blog-starter.vercel.app

Cómo usar

Puede elegir uno de los dos métodos siguientes para utilizar este repositorio:

Implementación con un solo clic

Implemente el ejemplo usando Vercel:

Clonar y Implementar

Ejecute create-next-app con pnpm para iniciar el ejemplo:

pnpm create next-app --example https://github.com/vercel/examples/tree/main/solutions/blog blog

Luego, ejecute Next.js en modo de desarrollo:

pnpm dev

Impléndelo en la nube con Vercel (Documentación).
